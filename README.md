# Bitcoin Thieves' Lottery

University Project.

## Setup project

```shell
wget https://tanguy.s3.fr-par.scw.cloud/projects/bitcoin-thieves-lottery/utxos.csv.gz
gzip -d utxos.csv.gz

python3 -m venv .venv
source .venv/bin/activate  # On entre dans l'env

pip3 install -r requirements.txt

wget https://tanguy.s3.fr-par.scw.cloud/projects/bitcoin-thieves-lottery/utxos.csv.gz
```
